VidMake Documentation

Script-based video maker using .VidMake JSON files.

1. Overview

VidMake turns JSON description files (.VidMake) into rendered MP4 videos. The initial version focuses on timed text overlays on a solid background.

2. File Format: .VidMake

A .VidMake file is a JSON object with these top-level fields:

FieldTypeRequiredDescription
resolution[width, height]noVideo resolution (default [1280, 720])
fpsnumbernoFrames per second (default 30)
durationnumberyesTotal duration of the video in seconds
background_colorstringnoHex color for background (default #000000)
video_codecstringnoFFmpeg video codec (default libx264)
audio_codecstringnoFFmpeg audio codec (default aac)
textsarraynoList of text events

2.1 Text event objects

Each entry in texts is an object:

FieldTypeRequiredDescription
textstringyesText to render
startnumbernoStart time (seconds, default 0)
endnumbernoEnd time (seconds, default = duration)
xnumber or "center"noX position (pixels or "center")
ynumber or "center"noY position (pixels or "center")
fontsizenumbernoFont size (default 48)
colorstringnoText color (named or hex, default #FFFFFF)

2.2 Full example

{
  "resolution": [1920, 1080],
  "fps": 30,
  "duration": 10,
  "background_color": "#101827",
  "video_codec": "libx264",
  "audio_codec": "aac",
  "texts": [
    {
      "text": "VidMake Intro",
      "start": 0,
      "end": 3,
      "x": "center",
      "y": "center",
      "fontsize": 72,
      "color": "#38bdf8"
    },
    {
      "text": "Script-based video in JSON",
      "start": 3,
      "end": 7,
      "x": "center",
      "y": 600,
      "fontsize": 40,
      "color": "#e5e7eb"
    }
  ]
}

3. CLI Usage

pip install -r requirements.txt
python vidmake.py my-video.VidMake out.mp4

Requirements:

  • Python 3.9+
  • ffmpeg installed and on PATH
  • moviepy (installed via requirements.txt)
