VidMake

Describe videos in text. Generate them from the command line.

What is VidMake?

VidMake is a script-based video maker. You write .VidMake files in JSON to describe resolution, background color, and timed text overlays, then render them to MP4.

Quick Start

  1. Install Python 3.9+, ffmpeg, and moviepy.
  2. Create a example.VidMake file.
  3. Run python vidmake.py example.VidMake out.mp4.
{
  "resolution": [1280, 720],
  "fps": 30,
  "duration": 8,
  "background_color": "#20232A",
  "texts": [
    {
      "text": "VidMake Demo",
      "start": 0,
      "end": 3,
      "x": "center",
      "y": "center",
      "fontsize": 64,
      "color": "#61DAFB"
    }
  ]
}
